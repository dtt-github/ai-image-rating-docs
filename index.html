API Reference

POST /api/v1/token Get an access token

Exchange your client credentials for a JWT bearer token (valid 1 hour).

curl -X POST https://api.yourserver.com/api/v1/token \ -H "Content-Type: application/json" \ -d '{"client_id": "agency_...", "client_secret": "sk_..."}'
POST /api/v1/rate Rate an uploaded image

Upload an image file for aesthetic analysis. Returns score, tags, content rating, quality, and portfolio gate decision.

curl -X POST https://api.yourserver.com/api/v1/rate \ -H "Authorization: Bearer YOUR_TOKEN" \ -F "file=@photo.jpg"
POST /api/v1/rate-url Rate an image from URL

Pass a public image URL for analysis. Same response format as file upload.

curl -X POST https://api.yourserver.com/api/v1/rate-url \ -H "Authorization: Bearer YOUR_TOKEN" \ -H "Content-Type: application/json" \ -d '{"url": "https://example.com/photo.jpg"}'
GET /api/v1/usage Check your quota

Returns your current usage and remaining requests for the day.

curl https://api.yourserver.com/api/v1/usage \ -H "Authorization: Bearer YOUR_TOKEN" # Response: {"used": 42, "limit": 1000, "remaining": 958}
POST /api/v1/feedback Submit feedback

Help improve the AI by submitting your rating (0-3) for any analyzed image.

curl -X POST https://api.yourserver.com/api/v1/feedback \ -H "Authorization: Bearer YOUR_TOKEN" \ -H "Content-Type: application/json" \ -d '{"image_hash": "a1b2c3...", "user_score": 3}'

Full interactive documentation available at /api/v1/docs (Swagger UI) and /api/v1/redoc (ReDoc) on the API server.

Quick Start

import requests BASE = "https://api.yourserver.com/api/v1" # 1. Get token token_resp = requests.post(f"{BASE}/token", json={ "client_id": "agency_abc123def456", "client_secret": "sk_your_secret_here", }) token = token_resp.json()["access_token"] headers = {"Authorization": f"Bearer {token}"} # 2. Rate an image with open("photo.jpg", "rb") as f: resp = requests.post(f"{BASE}/rate", headers=headers, files={"file": f}) result = resp.json() print(f"Score: {result['score']}, Rating: {result['rating']}") print(f"Tags: {[t['tag'] for t in result['tags']]}") print(f"Portfolio suitable: {not result['rejected']}")

Rate an image from URL const resp = await fetch(`${BASE}/rate-url`, { method: "POST", headers: { Authorization: `Bearer ${access_token}`, "Content-Type": "application/json", }, body: JSON.stringify({ url: "https://example.com/photo.jpg" }), }); const result = await resp.json(); console.log(`Score: ${result.score}, Rating: ${result.rating}`);
